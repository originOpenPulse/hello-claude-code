Architecture Snapshot

项目结构速览

这是一个运行在 Bun 上的终端 Agent，而不是浏览器产品。核心是 React + Ink UI 和一套多轮工具调用状态机。

主执行链

  1. CLI 入口解析参数并选择运行模式。
  2. REPL 或 QueryEngine 组装会话状态。
  3. query() 驱动模型请求、流式响应与工具回流。
  4. tools.ts 按权限上下文装配工具池。
  5. services/api/claude.ts 负责 provider 路由与请求构造。

高频专题

  • 上下文治理：snip、microcompact、autocompact、context collapse
  • 扩展总线：skills、plugins、MCP、hooks
  • 长会话系统：prompt runtime、session storage、memory
  • 远程协作：Bridge、agents、tasks、remote session

Project Context

这页在解决什么问题

统一阅读入口以 deep_dive/ 为准，历史视角目录保留为参考材料。
